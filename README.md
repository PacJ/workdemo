### 코드(code)

#### 1)인라인 코드(inline code)

- 벡틱(`)으로 강조할 내용을 감싼다.
  - repository에서 프로젝트의 설명을 부여해줄때 `README.md`를 사용한다.

- 벡틱(`) 3개로 html, css, java등 코드를 작성할때 사용한다.

```

System.out.println("Hello Java");
}
```
